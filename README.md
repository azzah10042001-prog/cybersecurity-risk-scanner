# cybersecurity-risk-scanner
AI-powered cybersecurity awareness tool for phishing and digital threat detection

This project focuses on improving human cybersecurity awareness by translating complex threat signals into simple, actionable insights for non-technical users.

---

## 📌 Key Features

- 🔍 Threat level classification (Safe → Critical)
- 🧠 AI-based attack type detection
- ⚠️ Suspicious indicator highlighting
- 🗣️ Simple explanation for non-technical users
- 📋 Step-by-step recommended actions
- 🎓 Cybersecurity awareness education layer
- 📊 Confidence score estimation

---

## 🧩 Supported Attack Types

- Phishing
- Smishing
- Vishing
- Business Email Compromise (BEC)
- Malware Delivery
- Credential Harvesting
- Tech Support Scam
- Fake Job Scam
- Investment Scam
- Identity Theft
